## BuildX Suite

A suite of built-environment modules covering the full building lifecycle —
from city context and site through architecture, structure, energy, and
infrastructure. Each module is self-contained with its own backend and
frontend, unified under a single suite shell.

## Modules

| Module         | Status  | Purpose                                                           |
|----------------|---------|------------------------------------------------------------------|
| City           | live    | 3D city context from CityGML / CityJSON ingestion                |
| Architecture   | live    | Generative floor plans and massing; IFC / DXF / PDF import       |
| Structure      | live    | Structural analysis with permit-grade verification               |
| Energy         | live    | Building energy performance and certificates                     |
| Roads          | live    | Generative road design over terrain with cost & carbon estimates |
| Project Mgmt   | live    | Cross-module project aggregation and reporting                   |

Modules share a common shell and data model, so a project flows from one
discipline to the next without leaving the suite.
